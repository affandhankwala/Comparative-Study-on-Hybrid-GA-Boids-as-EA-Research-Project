## Abstract
Optimization of continuous real-value functions involves minimizing a function over a specific landscape. Evolutionary computation and swarm intelligence offer an efficient solution to handle these complex environments through established methods such as genetic algorithms (GAs) and particle swarm optimizations (PSOs).
